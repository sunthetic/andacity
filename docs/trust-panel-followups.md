# Trust Panel Follow-ups

Additional backend fields that would let the shared trust panel expose cleaner decision support:

- Hotels
  Room-level cancellation cutoff timestamps and payment timing per rate plan would let the detail page replace broad policy blurbs with exact room terms.

- Hotels
  Mandatory fee breakdown fields, including resort fee amount and tax inclusions, would let the price summary move from "estimated fees" to a clearer payable-total breakdown.

- Cars
  Offer-level deposit amounts, mileage limits, and counter-fee breakdowns would make the rental trust panel more precise than the current policy summary copy.

- Cars
  Pickup-counter hours, after-hours return rules, shuttle requirements, and explicit airport-vs-city location fields would let the new detail-page decision summary expose stronger location and constraint context.

- Flights
  Fare-rule summary fields for refund penalties, change penalties, and cancellation deadlines are still missing, so the flight panel can only show refundable/changeable flags today.

- Flights
  Carry-on inclusion, seat-selection fee expectations, and tax-and-fee totals per itinerary would let the flight price summary expose a more trustworthy checkout estimate.

- Hotels
  Landmark-distance fields, neighborhood centroids, and room-level fee flags would let the new detail-page decision summary move beyond address-only location context and broad fee caveats.
